Conversión de Presión a Número de Onda

Funcionamiento Detallado

Configuración del Láser

Seleccione la longitud de onda del láser utilizado en su experimento:

  • 488nm: Láser de argón-ion
  • 514.5nm: Estándar dorado para ruby
  • 532nm: Láser de estado sólido
  • 633nm: Láser HeNe
  • 785nm: Para mediciones especiales

Entrada de Presión

Ingrese el valor de presión en Gigapascales (GPa):

  • Rango válido: 0 a 500 GPa
  • Use punto o coma como separador decimal
  • El cálculo se realiza automáticamente

Características Avanzadas

Ajuste de Calibración

Compense variaciones instrumentales con el ajuste fino:

  1. Seleccione el láser deseado
  2. Use los botones + y - para ajustar
  3. El valor se guarda automáticamente por láser

Nota: El ajuste predeterminado es 0.0 cm⁻¹

Conversión de Número de Onda a Presión

Funcionamiento Detallado

Entrada de Número de Onda

Ingrese el número de onda Raman en cm⁻¹:

  • El rango válido varía según el láser seleccionado
  • Para láser 785nm, se aceptan valores negativos
  • El cálculo se realiza automáticamente

Validación Automática

La aplicación verifica automáticamente:

  • Si el valor está dentro del rango posible
  • Si resultaría en presión válida (0-500 GPa)
  • Muestra mensajes claros para valores inválidos

Gestión de Datos

Historial de Cálculos

  • Almacena hasta 50 conversiones recientes
  • Formato: Presión (GPa) → Número de Onda (cm⁻¹)
  • Limpieza automática al reiniciar la app

Importación/Exportación

  • Exportar: Guarda en archivo .txt en la carpeta Descargas
  • Importar: Carga datos de archivos .txt anteriores
  • Formato compatible con análisis en otros softwares
